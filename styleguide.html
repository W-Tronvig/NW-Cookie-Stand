MAIN

For the main portion of the page, use the following CSS:
display: flex;

For the nav, section, aside portions of the page, use the following CSS:
vertical-align: top;

For the nav and aside portions of the page, use the following CSS:
flex-grow: 1;
flex-shrink: 0;
flex-basis: 320;
flex: 1 0 320px;
height: 100px;
border-radius: 20px;
margin: 0 35px;

This will allow the section, aside, and nav elements to grow and shrink correctly when resizing the page.
